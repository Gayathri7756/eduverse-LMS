## Quick Start (5 Minutes)

### Install
```bash
cd frontend && npm install
```

### Start Backend
```bash
cd backend && npm start
```

### Start Frontend (New Terminal)
```bash
cd frontend && npm run dev
```

### Open Browser
```
http://localhost:5173
```

# 🚀 Quick Certificate Start

## 3 Simple Steps

### Step 1: Get Your User ID (30 seconds)
```
Open: get-my-uid.html
Click: 🔍 Extract My User ID
Copy: Your User ID
```

### Step 2: Complete Course (1 minute)
```
Open: simple-complete-course.html
Paste: Your User ID
Select: A course
Click: ✅ Complete Course & Generate Certificate
Wait: Success message
```

### Step 3: View Certificate (30 seconds)
```
Go to: http://localhost:5173
Click: My Learning
Refresh: Page (Ctrl+R)
Click: 🎓 Generate Certificate button
View: Your certificate!
```

## File Locations

All files are in your project root:
- `get-my-uid.html` - Extract your User ID
- `simple-complete-course.html` - Complete course & generate certificate
- `CERTIFICATE_TESTING_GUIDE.md` - Detailed guide
- `CERTIFICATE_SETUP_COMPLETE.md` - Full documentation

## What Happens

When you click the button:
1. ✅ All videos marked as watched
2. ✅ All quizzes marked as passed (100%)
3. ✅ All assignments marked as submitted (100%)
4. ✅ Certificate generated with unique number
5. ✅ Certificate button appears in My Learning

## Certificate Shows

- 🏆 Trophy icon
- Your name
- Course name
- Completion date
- Certificate number
- Instructor name

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "No token found" | Log in at http://localhost:5173 first |
| "No courses" | Make sure backend is running |
| "Not enrolled" | Enroll in course first at http://localhost:5173 |
| "Course not completed" | Tool shows what's missing - all must be 100% |

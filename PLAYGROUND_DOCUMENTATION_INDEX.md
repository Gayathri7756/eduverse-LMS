## Key Information

### Supported Languages (16)
JavaScript, Python, Java, C++, C, C#, Go, Rust, Ruby, PHP, Swift, Kotlin, TypeScript, R, Bash, SQL

### Setup Time
- Get API key: 2 minutes
- Add to .env: 1 minute
- Restart backend: 1 minute
- Test: 1 minute
- **Total: 5 minutes**

### Free Tier
- 100 requests per day
- 20 requests per minute
- Perfect for development

### Performance
- JavaScript: Instant (< 100ms)
- Other languages: 1-5 seconds
- Timeout: 20 seconds max

### Security
- Isolated code execution
- No filesystem access
- API key in .env
- Input validation
- Safe error messages

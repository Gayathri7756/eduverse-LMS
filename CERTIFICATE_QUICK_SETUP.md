# 🚀 Certificate Setup - Quick & Easy

## 3 Simple Steps

### Step 1: Get Your User ID (Inside Your App)
```
1. Go to: http://localhost:5173/get-user-id
2. Click: 📋 Copy to Clipboard
3. Your ID is now copied!
```

### Step 2: Complete Course
```
1. Open: simple-complete-course.html
   (Double-click in File Explorer or use the file path)
2. Paste: Your User ID
3. Select: A course
4. Click: ✅ Complete Course & Generate Certificate
5. Wait: Success message
```

### Step 3: View Certificate
```
1. Go to: http://localhost:5173/my-learning
2. Refresh: Page (Ctrl+R)
3. Click: 🎓 Generate Certificate button
4. View: Your certificate!
```

## That's It! 🎉

Your certificate is now ready to view and share.

---

**Why this works:**
- The GetUserID page runs inside your app, so it can access your login token
- The simple-complete-course.html tool marks all course content as complete
- The certificate is generated automatically
- Your progress is saved to the database

**Need help?** Check the browser console (F12) for any errors.

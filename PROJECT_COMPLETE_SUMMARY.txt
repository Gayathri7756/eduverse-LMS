================================================================================
                          TROUBLESHOOTING
================================================================================

Backend won't start:
  $ lsof -i :5000
  $ kill -9 <PID>
  $ npm run dev

Frontend won't start:
  $ rm -rf node_modules package-lock.json
  $ npm install
  $ npm run dev

Features not working:
  - Hard refresh: Ctrl+Shift+R
  - Check browser console: F12
  - Check backend: http://localhost:5000/health

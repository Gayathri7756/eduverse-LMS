## 🎨 Theme Specifications Applied

### Color Palette (All Pages)
```
Primary (Warm Taupe):
  - Primary-900: #5a5247 (Dark text)
  - Primary-700: #8a7968 (Secondary text)
  - Primary-600: #9d8b7a (Buttons)
  - Primary-500: #b5a394 (Accents)

Secondary (Soft Grey):
  - Secondary-600: #b5a89d (Hover states)
  - Secondary-500: #c9bfb5 (Secondary elements)

Neutral/Background:
  - Neutral-100: #fdfaf6 (Light background)
  - Secondary-100: #faf8f6 (Gradient component)
```

### Glass Effects (All Pages)
```
.glass - Standard frosted glass
  - 25% white opacity
  - 25px blur
  - Inset shadow for depth

.glass-dark - Taupe-tinted glass
  - 15% taupe opacity
  - 25px blur
  - For headers/sections

.glass-secondary - Grey-tinted glass
  - 12% grey opacity
  - 25px blur
  - For secondary elements
```

### Background Gradient (All Pages)
```
linear-gradient(135deg, #e8e4e0 0%, #ddd5cc 25%, #d9d3cc 50%, #e3dcd3 75%, #e8e4e0 100%)
```

### Styling Consistency (All Pages)
- Rounded corners: `rounded-xl` (12px) for inputs/buttons, `rounded-2xl` (16px) for cards
- Text colors: `text-primary-900` for headings, `text-primary-700` for body
- Button styling: `bg-primary-600 hover:bg-primary-700`
- Border styling: `border-primary-400/20`
- Transitions: `transition` on all interactive elements

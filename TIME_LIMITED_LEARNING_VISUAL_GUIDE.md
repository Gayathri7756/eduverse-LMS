## Quiz Submission Flow

```
1. Student clicks "Start Quiz"
   ↓
2. Quiz interface opens with timer (30 min for Quiz 1)
   ↓
3. Student answers all questions
   ↓
4. Student clicks "Submit Quiz"
   ↓
5. System auto-scores (70% = pass)
   ↓
6. Results shown:
   - Score: 85%
   - Status: ✅ Quiz Passed!
   - Detailed review of answers
   ↓
7. Student can proceed to next quiz or assignment
```

## Assignment Submission Flow

```
1. Student clicks "View Assignment"
   ↓
2. Assignment details shown:
   - Description
   - Instructions
   - Due date
   - Max score
   ↓
3. Student enters submission text
   ↓
4. Student clicks "Submit Assignment"
   ↓
5. Submission confirmed:
   - ✅ Assignment Submitted!
   - Submission details shown
   - Timestamp recorded
   ↓
6. Instructor reviews and grades
```

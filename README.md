# Learning Management System (LMS)

A full-stack LMS built with React, Vite, TailwindCSS, Node.js, Express, and Firebase.

## Project Structure

```
lms/
├── frontend/          # React + Vite + TailwindCSS
├── backend/           # Node.js + Express
└── README.md
```

## Features

- User authentication (student and instructor roles)
- Landing page with featured courses
- Course catalog with search and filters
- Course detail page
- Instructor dashboard to create courses
- Student dashboard showing enrolled courses

## Setup Instructions

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

## Environment Variables

Create `.env` files in both frontend and backend directories with required Firebase and API configurations.
